# Unidad 2: Componentes y Librerías

## Descripción General
En esta unidad se estudia la arquitectura modular del software. Se aprende a crear, empaquetar y distribuir componentes reutilizables, siguiendo los estándares de la comunidad de Python.

## Temario

1.  **[2.1 Definición conceptual de componentes](2.1.md)**
    *   Módulos y paquetes.
2.  **[2.2 Uso de librerías del lenguaje (StdLib)](2.2.md)**
    *   Aprovechamiento de la biblioteca estándar de Python.
3.  **[2.3 Creación de componentes propios (OOP)](2.3.md)**
    *   Clases modulares y mixins.
4.  **[2.4 Creación de paquetes y librerías (PyPI)](2.4.md)**
    *   Empaquetado profesional y distribución de código.

## Objetivos de Aprendizaje
Comprender y aplicar los principios de modularidad y reutilización para construir sistemas escalables y mantenibles.
