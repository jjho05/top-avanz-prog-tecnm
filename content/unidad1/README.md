# Unidad 1: Interfaces Gráficas de Usuario

## Descripción General
Esta unidad se enfoca en el desarrollo de interfaces modernas y responsivas utilizando CustomTkinter. Se exploran desde los fundamentos de la arquitectura Tcl/Tk hasta el manejo avanzado de eventos y el diseño de componentes personalizados.

## Temario

1.  **[1.1 Interfaz Gráfica de Usuarios (Arquitectura)](1.1.md)**
    *   Ciclo de vida de una aplicación GUI.
    *   El bucle principal (`mainloop`).
2.  **[1.2 Tipos de Eventos](1.2.md)**
    *   Eventos de mouse, teclado y ventana.
3.  **[1.3 Manejo de Eventos (Binding)](1.3.md)**
    *   Enlace de funciones y lambdas.
4.  **[1.4 Componentes de Control y Diálogos](1.4.md)**
    *   Widgets avanzados y ventanas modales.

## Objetivos de Aprendizaje
Ser capaz de diseñar, implementar y depurar aplicaciones de escritorio con una experiencia de usuario moderna y profesional.
