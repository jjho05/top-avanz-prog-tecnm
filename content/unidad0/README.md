# Unidad 0: Fundamentos del Ecosistema Python

## Descripción General
Esta unidad introductoria establece las bases profesionales para el desarrollo en Python. A diferencia de cursos básicos, aquí nos enfocamos en el "Professional Tooling": entornos virtuales, calidad de código y estructura de proyectos escalables.

## Temario

1.  **[0.1 Configuración del Entorno](0.1_python_env.md)**
    *   Instalación de Python 3.10+.
    *   Configuración de VS Code como IDE avanzado.
    *   Extensiones esenciales (Pylance, Black).

2.  **[0.2 Entornos Virtuales](0.2_virtual_envs.md)**
    *   El problema del "Dependency Hell".
    *   Uso de `venv` y `pip`.
    *   Gestión de `requirements.txt`.

3.  **[0.3 Repaso Acelerado de Python](0.3_python_review.md)**
    *   Estructuras avanzadas (Sets, Tuples, Dictionaries).
    *   POO moderna (`@property`, `__dunder__`, `super()`).
    *   Decoradores y Callbacks.

4.  **[0.4 Calidad de Código](0.4_code_quality.md)**
    *   Estándar PEP 8.
    *   Linters (Flake8) y Formatters (Black).
    *   Type Hinting (Mypy).

5.  **[0.5 Estructura de Proyecto](0.5_project_structure.md)**
    *   Src Layout.
    *   Separación de responsabilidades.
    *   Gitignore y configuración.

## Objetivos de Aprendizaje
Al finalizar esta unidad, el estudiante tendrá un entorno de desarrollo profesional configurado y será capaz de escribir código Python que cumpla con los estándares de la industria.
