# Unidad 3: Programación Concurrente

## Descripción General
Esta unidad aborda el procesamiento simultáneo mediante hilos y procesos. Se analizan los retos de la concurrencia, desde el GIL de Python hasta la sincronización de recursos compartidos.

## Temario

1.  **[3.1 Hilos vs Procesos y GIL](3.1.md)**
    *   Diferencias fundamentales y el Global Interpreter Lock.
2.  **[3.2 I/O Bound vs CPU Bound](3.2.md)**
    *   Identificación de cuellos de botella y elección de técnica.
3.  **[3.3 Creación y control de hilos (API)](3.3.md)**
    *   Manejo de la librería threading y ciclo de vida de hilos.
4.  **[3.4 Sincronización (Locks, Queues)](3.4.md)**
    *   Prevención de condiciones de carrera y comunicación entre hilos.

## Objetivos de Aprendizaje
Programar aplicaciones capaces de ejecutar múltiples tareas de forma eficiente, mejorando la responsividad y el rendimiento.
