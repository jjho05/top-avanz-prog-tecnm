# Unidad 4: Acceso a Datos

## Descripción General
Se exploran las técnicas modernas de persistencia de datos. Desde el uso de SQL puro hasta la implementación de ORMs sofisticados y la visualización de grandes volúmenes de información.

## Temario

1.  **[4.1 Persistencia y Modelo Relacional](4.1.md)**
    *   Estructura de bases de datos y principios ACID.
2.  **[4.2 Conexión a BD (SQLAlchemy)](4.2.md)**
    *   Configuración de motores y mapeo relacional.
3.  **[4.3 CRUD y Seguridad](4.3.md)**
    *   Operaciones de datos y prevención de inyecciones SQL.
4.  **[4.4 Visualización de Datos (Pandas)](4.4.md)**
    *   Análisis y presentación gráfica de información técnica.

## Objetivos de Aprendizaje
Integrar sistemas de bases de datos persistentes y seguros en aplicaciones de escritorio y herramientas de análisis.
