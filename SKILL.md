# Tópicos Avanzados de Programación (SCD-1027)

Unidad de aprendizaje curricular ("Skill") diseñada para estudiantes de Ingeniería en Sistemas Computacionales del Tecnológico Nacional de México (TecNM).
Este repositorio contiene el material teórico extendido ("Doble Enciclopedia Técnica") y práctico para la materia, garantizando una profundidad de más de 800 líneas por tema.

## Estructura del Curso

### [Unidad 0: El Ecosistema Profesional](content/unidad0/README.md)
Fundamentos que separan a un aficionado de un ingeniero.
*   **0.1** Configuración del Entorno (Python 3.10+, VS Code).
*   **0.2** Entornos Virtuales y Gestión de Dependencias (venv, pip, poetry).
*   **0.3** Python Avanzado (Decoradores, Generadores, Match Case).
*   **0.4** Calidad de Código (PEP 8, Linters, Type Hints).
*   **0.5** Estructura de Proyectos (Src Layout).

### [Unidad 1: Interfaces Gráficas de Usuario](content/unidad1/README.md)
Desarrollo de escritorio moderno con **CustomTkinter**.
*   **1.1** Arquitectura GUI (Tcl/Tk, Mainloop).
*   **1.2** Sistema de Eventos (Mouse, Teclado, Virtuales).
*   **1.3** Manejo Avanzado (Binding, Debouncing, Lambdas).
*   **1.4** Widgets de Control y Diálogos.

### [Unidad 2: Componentes y Librerías](content/unidad2/README.md)
Arquitectura de software modular.
*   **2.1** Conceptos: Módulos, Paquetes, Cohesión.
*   **2.2** La Librería Estándar (Pathlib, Collections, Json).
*   **2.3** Componentes Propios (OOP, Mixins, Slots).
*   **2.4** Empaquetado y Distribución (PyPI, Wheels, pyproject.toml).

### [Unidad 3: Programación Concurrente](content/unidad3/README.md)
Multitarea y paralelismo.
*   **3.1** Hilos vs Procesos (GIL, Context Switching).
*   **3.2** Clasificación I/O Bound vs CPU Bound.
*   **3.3** API Threading (Lifecycle, Daemon).
*   **3.4** Sincronización (Locks, Semáforos, Colas Productor-Consumidor).

### [Unidad 4: Acceso a Datos](content/unidad4/README.md)
Persistencia y Ciencia de Datos básica.
*   **4.1** Modelo Relacional y ACID (SQLite).
*   **4.2** ORM con SQLAlchemy 2.0.
*   **4.3** Operaciones CRUD y Seguridad (SQL Injection).
*   **4.4** Visualización (Pandas, Matplotlib, Seaborn).

### [Unidad 5: Programación Móvil](content/unidad5/README.md)
Desarrollo Cross-Platform.
*   **5.1** Python en Móvil (Kivy, BeeWare, Flet).
*   **5.2** Nativas vs Híbridas vs PWA.
*   **5.3** Arquitectura Android (Lifecycle, Permisos).
*   **5.4** Desarrollo con **Flet** (Flutter-based).
*   **5.5** Seguridad Móvil (OWASP Mobile Top 10).

## Stack Tecnológico
*   **Lenguaje:** Python 3.10+
*   **GUI:** CustomTkinter (Desktop), Flet (Mobile).
*   **Datos:** SQLAlchemy, SQLite, Pandas.
*   **Gráficos:** Matplotlib.
*   **Herramientas:** VS Code, Black, Flake8, Git.
