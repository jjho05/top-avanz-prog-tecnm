# Contribuyendo

¡Gracias por tu interés en mejorar este material educativo del TecNM!

## Flujo de Trabajo

1.  Haz un **Fork** del repositorio.
2.  Crea una rama (`git checkout -b feature/nueva-unidad`).
3.  Haz tus cambios (Asegúrate que el código Python siga PEP-8).
4.  Haz **Commit** (`git commit -m 'Mejora en ejemplo de hilos'`).
5.  Haz **Push** (`git push origin feature/nueva-unidad`).
6.  Abre un **Pull Request**.

## Estándares

- Todo el código debe estar comentado en Español.
- Los nombres de variables deben ser descriptivos (`contador_usuarios` en vez de `c`).
